## 2. 商品ラインナップ戦略パターン

### 「最初の1本」の選定パターン

AIカテゴリで売れているトップ販売者の初期商品を分析すると、以下の3パターンに集約される。

**A. ノウハウ実録型**（最多・優太に最適）
- 「私はこうして〇万円稼いだ」という実録形式
- 例: まな「10ヶ月連続100万円達成した僕の、AI×note収益化の全ノウハウ」¥22,980
- 例: こうめい「市場から逆算する売れるジャンルの見つけ方〜8日で0→1達成した方法〜」¥1,980
- 特徴: 著者の体験が核なので規約クリアしやすく、差別化しやすい

**B. 教科書・マニュアル型**（シリーズ展開に有利）
- 「○○の教科書」「完全ガイド」という包括的なリファレンス形式
- 例: しゅうへい「世界一わかりやすいClaudeアプリの教科書」¥2,980

- 特徴: ツール名＋「教科書」の組み合わせがベスト。シリーズ化しやすい

**C. ツール型・システム型**（高単価に向く）
- 特定ツールの使い方ではなく「仕組み全体」を提供
- 例: ベク「note×Claude Code全自動マネタイズ」¥1,280

## 5. 誇大表現のセーフ/アウト境界線

Brain公式ガイドライン（最終更新2026/02/25）の第4項「誇大広告にあたる表現・事実と誤認させるような表現」を元に整理。

### セーフライン（公式OKとなる表現）

- **「私は〜を達成した」（自分の体験として明示）**
  → 例:「2ヶ月で月14万円を達成しました」
- **「個人の体験として〜」という文脈での実績提示**
  → 例:「私の実体験をそのまま書いた記録です」
- **再現可能性を断言しない形での体験談**
  → 例:「私はこの手順で稼ぐことができました」
- **数字を出す場合でも主語が「私は」の形**
  → 例:「私がやったらこうなった、という記録です」

### アウトライン（ガイドライン明示の禁止表現）

- **「誰でも儲かる」「誰でも再現できる」**
- **「100%成功する」「再現性100%」**
- **「必ず〜できる」「絶対に〜なる」**
- **「不労所得」という言葉（大げさ・虚偽とみなされる）**
- **「短時間で簡単に稼げる」（根拠が必要）**
- **「知らないと損をする」「人生が変わる」（根拠なし）**
- **「○○が稼げる」「○○を稼ぐ方法」（読者全員が達成できなければNG）**
- **「世の中の99%の人が知らない」「唯一の方法」**
- **「低学歴でもできる」（差別的表現）**
- **「1日15分の作業で簡単に月10万円稼げる」（根拠なし）**

### グレーゾーンでよく使われる表現（実際に通過している商品より）

実際に審査通過して販売されている商品のタイトル・プロフィールで確認された表現:
- **「月○万円稼ぐ方法」→ 実際には「達成した僕の、全ノウハウ」と主語を明確化して回避**（まなの商品例）
- **「初心者でも〜できる」→「初心者向け」という修飾表現ではなく「初心者でもわかる解説」**という教育文脈に寄せて回避している例が多い
- **「月20万狙える」「月10万を目指す」→「○○を目指す」「〜狙える可能性」**という希望表現は一定数通過している模様（ただし正確には規約グレー）
- **「0→1」「収益ゼロから」→ 結果でなく「変化の過程」という文脈で使われており通過例あり**
- **「最速」「完全攻略」「完全版」→ 大げさだが内容が伴っていれば通過の傾向**

### 優太向け推奨表現

「月14万円達成」という事実を使う際は以下のように書く:
- セーフ: 「私が2ヶ月で月14万円を達成したリアルな記録です」
- セーフ: 「文系大学生の私が実際にやった全手順を公開します」
- アウト: 「やれば誰でも月14万円になれます」
- アウト: 「確実に稼げるようになります」

## 6. 優太向けカスタム推奨（最終提案）

### プロフィール設計

**推奨: パターンA（実績訴求型）ベース + パターンC（ニッチ特化）のポジション宣言を組み合わせ**

```
文系大学生がAIだけで2ヶ月で月14万円を達成しました。
Claude Code × クラウドワークスで、非エンジニアが案件受注→納品できる
リアルな記録を全部公開します。プログラミング経験ゼロからのスタートです。
Xで毎日AI副業の実録を発信中↓
```

理由:
1. 「文系大学生」×「2ヶ月」×「月14万円」は具体的で差別化できる数字組み合わせ

3. 「プログラミング経験ゼロ」という属性が非エンジニアの購入者に安心感を与える
4. 「リアルな記録を全部公開」という表現がガイドラインをクリアしつつ信頼感を構築
