## エグゼクティブサマリー

### Brain サムネの 3 大パターン

**パターンA: 情報商材ゴールド型（最多・約40%）**
- 黒/ダーク宇宙 + 金ピカグラデーションテキスト
- 「完全攻略」「教科書」「バイブル」などの強い動詞
- 金のロゴマークバッジ・特典数アイコン（「豪華8大特典付き」等）
- 代表例: Antigravityの教科書、note×Claude Code全自動マネタイズ、AI占い完全攻略マニュアル

**パターンB: ゲーム/アニメ + ド派手ネオン型（約30%）**
- 黒背景 + 虹色/ネオンカラーの光線エフェクト
- アニメキャラクター・イラストアイコンを使用
- 「月○万円」の大きな数字をオレンジ/ピンクで配置
- 代表例: GoogleAI×クラウドワークス（メラメラくん）、AI×ライティング完全攻略

**パターンC: テキスト密度型（約25%）**
- 濃紺/ダーク背景 + ホワイトテキスト多段
- ツール名（n8n、Claude、Antigravity等）を大きく配置
- 箇条書き・チェックマーク・矢印でメリットを羅列
- 代表例: n8n完全攻略バイブル、OpenClawで〜AI社員を雇う入門

**例外パターンD: ライト系クリーン型（約5%）**
- オフホワイト/ライトオレンジ背景
- ツールロゴを大きく、テキストは最小限
- 顔写真（丸型切り抜き）あり
- 代表例: Claudeアプリの教科書（しゅうへい）、NanoBanana Pro×クラウドワークス

### note との最大の違い 5 項目

1. **背景色**: note → オフホワイト/ライトグレー　Brain → ほぼ全件ダーク系（黒/濃紺/宇宙）
2. **フォント**: note → モダンゴシック・スリム　Brain → 超極太ゴシック・影付き・グラデーション
3. **数字訴求**: note → 数字は控えめ or なし　Brain → 「月14万」「100万円」を激デカで必ず配置
4. **アイコン**: note → 幾何学装飾　Brain → ゲームUI風バッジ・炎エフェクト・光線
5. **色数**: note → 2〜3色制限でシンプル　Brain → 5〜8色のド派手多色展開

### 優太向け推奨ライン

「情報商材感は抑えつつ、Brainの数字訴求・ダーク系の文法に乗る」ポジション。
具体的には「パターンCのテキスト密度型」をベースに、情報商材のギラギラを抑えたクリーンダーク路線が差別化ポイントになる。

## 5. NG リスト（避けるべきデザイン要素）

### Brain相場に溶け込みすぎてしまうNG（差別化のため避ける）

1. **金ピカグラデーションテキスト（ゴールド文字）**: Antigravity教科書、note×Claude Codeが使用済み。多用されすぎて埋もれる

### 優太のブランドイメージを損なうNG（ターゲット感のため避ける）

6. **過度な成人向け・同人誌感**: AI-01（大人の漫画）の路線。全ターゲットが逃げる
